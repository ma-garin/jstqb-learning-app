# Repository Context

このリポジトリは、個人制作の非公式な「QA基礎学習PWA」です。

## データ構成

- `js/questionsData.js`: 制作者による独自問題。`sourcePolicy: "original"` を必須とする。
- `js/topicMap.js`: 章・節・LOコード・Kレベル・参照URLだけを扱う学習範囲マップ。
- `js/glossaryData.js`: 新卒向けに独自作成した用語説明。
- `data/release_notes.csv`: アプリ自体の更新履歴。

## 権利安全ルール

- 公式PDF、ロゴ、公式問題、公式本文を保存しない。
- 公式文章の翻訳、要約コピー、近似コピーを教材として収録しない。
- 外部シラバスは話題の位置を確認するためだけに使う。
- 問題、選択肢、解説、教材本文は白紙から独自に作成する。
- 公開前に `docs/rights-safety-checklist.md` を確認する。
